EPA Website
Redesign

My Role: UX / UI Designer

Disciplines: Usability Testing, Heuristics, IA, Style Guide, Responsive Design UX / UI Designer

Tools used: 2023

THE CHALLENGE

EPA’s website users need a simple and clear navigation and information architecture to find information they need about environmental policy and regulations. EPA’s website contains large amounts of information organized inconsistently. Related topics are treated separately. Language is unfamiliar and uses technical jargon.

The website redesign would focus on reorganizing content, simplifying navigation, introducing visual elements for recognition and consistency, and rewritting some of the terminology using words or concepts familiar to the user.
